# Pochas

## Preparación
1. Poner las alubias en la cazuela y añadir agua al ras y un vaso de agua más
2. Poner ajo, cebolla, pimiento verde, tomate y un chorro generoso de aceite de oliva virgen
3. Cuando rompa a hervir, dejar a fuego lento 30-40 minutos. Se puede añadir el agua que necesite

Se puede separar la verdura (pimiento, cebolla, etc.) y triturarla, añadiéndola a las pochas
