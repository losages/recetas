# ☑️ Spaguetti con espárragos trigueros y gambas

## Ingredientes
- Espárragos trigueros
- Spaguetti
- Ajos
- Perejil
- Gambas
- Queso para pasta y pimienta (opcional)

## Preparación
1. Cocer los espárragos 10-15 min (cortados en pequeños trozos)
2. Cocer los spaghetti
3. Freír las gambas, ajos y espárragos
4. Mezclar todo y añadir perejil y aceite (si se quiere también queso y pimienta)
