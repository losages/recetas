# Tiramisú (Pili S.L.)

## Ingredientes
- 1/2 kg de nata para montar
- 250 mg de queso Philadelphia o Mascarpone
- 1 cuajada
- Bizcochos duros (estrechos)
- Café (unas 9 tazas de cafetera)
- 1 chorrito de Baileys
- Chocolate rayado (duro, para rayar con rayadora)

## Preparación
1. Montar la nata, añadir el queso y la cuajada  
2. Preparar café. Añadir un chorrito de Baileys  
3. Colocar una capa de bizcochos, previamente mojados en café (que no empapen demasiado)  
4. Hacer capas:  
    4.1 Capa de nata  
    4.2 Capa de bizcochos  
    4.3 Capa de nata  
    4.4 Capa de bizcochos  
    4.5 Capa de nata  
5. Chocolate rayado
