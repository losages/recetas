# Base de pizza

## Ingredientes
- 250 g de harina
- 1/2 sobre de levadura Royal
- 1/2 cucharadita de café de sal
- 2 cucharadas de aceite de oliva
- 125 cl de agua templada (unos 30ºC)

## Preparación
1. Mezclar harina, sal y levadura
2. Añadir sal y aceite y amasar 3 minutos hasta obtener una bola lisa
3. Calentar al horno (210 ºC) y dejar reposar la masa durante 20 minutos
4. Harinar la bola y extender la masa con el rodillo
