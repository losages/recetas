# Tarta de Manzana (Montse)

## Ingredientes
- 1 huevo
- 1/2 vaso de leche
- 1/2 vaso de harina
- 1/2 vaso de azúcar
- 1/2 sobre de levadura Royal

## Preparación
1. Batir todo junto en la batidora  
2. Rociar el molde con mantequilla  
3. Partir la manzana en cuartos y a capas finas  
4. Colocar:  
    4.1 Una capa de manzana  
    4.2 Una capa de masa  
    4.3 Una capa de manzana  
    4.4 Una capa de masa  
    4.5 Una capa de manzana  
5. Colocar al horno 45-50 minutos a 150-175ºC
6. Una vez enfriada, se puede poner mermelada de albaricoque o melocotón
